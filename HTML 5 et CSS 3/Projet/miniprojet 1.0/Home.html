Objectifs

L’objectif du mini-projet est d’implémenter un programme en C visant à reproduire le célèbre jeu « Memory ».

Description

Memory est un jeu se pratiquant à un joueur ou plus (deux dans notre cas) au tour par tour, dont le but est de collecter les paires correspondantes.
Au début une grille de lettres sera révélée pendant quelques secondes (5 secondes par exemple), puis toutes les lettres seront cachées. Un premier joueur saisit les coordonnées de deux cases de la grille ;

  • Si c'est la même lettre qui apparaît dans les deux cases indiquées par le joueur, le joueur gagne un point et ces deux cases seront révélées jusqu’à la fin du jeu. Ensuite, le même joueur joue un autre tour et en saisit à nouveau les coordonnées de deux cases.
  • Si les lettres dans les deux cases ne correspondent pas (les lettres sont différentes), les deux lettres seront révélées pendant une courte durée puis cachées une autre fois, et c'est le rôle de l’autre joueur de jouer en suivant les mêmes règles énoncées cidessous.

Celui qui a une bonne mémoire se souviendra de l'emplacement des lettres déjà retournées et pourra les utiliser lorsqu'il en aura besoin pour compléter une paire.
Le gagnant est celui qui accumule plus de point à la fin du jeu (celui qui a formé le plus de paires).

Indications

La grille peut être représentée par :

  • Ou bien deux tableaux à deux dimensions, l’un est remplies par les lettres que les joueurs doivent révéler, l’autre mémorise si une case donnée est révélé ou non (par exemple 1 : pour dire révélé et 0 pour dire caché).
  • Ou bien un seul tableau à deux dimensions, dont chaque cellule est unenregistrement contenant un champ contenant la lettre, et un champ indiquantsi la lettre doit être révélée ou cachée.

Concernant le nombre de joueurs, on pourra utiliser un bouclage sur un tableau à une dimension contenant les scores de joueurs.
